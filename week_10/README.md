# Week 10
## Reinforcement Learning

Lecture by [Kirill Khrylchenko](https://github.com/rita1302).

Seminar by [Artem Matveev](https://github.com/matfu-pixel)


Youtube:
* [Lecture](https://youtu.be/pL80qvtB3dY)
* [Seminar](https://youtu.be/If1uocYfq7c)

In Week 10, we conclude our discussion of graph-based models for recommender systems, focusing in particular on unsupervised graph learning and industry applications (e.g., Twitter and TikTok). We then introduce reinforcement learning, starting with the multi-armed bandit setting and covering core concepts and basic algorithms.

During the seminar, we discuss the following papers from the YouTube team on bandits in recommender systems:
1. [Online Matching: A Real-time Bandit System for Large-scale Recommendations](https://arxiv.org/abs/2307.15893)
2. [Long-Term Value of Exploration: Measurements, Findings and Algorithms](https://dl.acm.org/doi/10.1145/3616855.3635833)
